## 🛠 核心技術標準 / Technical Standards

身為專業的落地頁開發者，我確保每一個專案都符合以下標準：
- **行動優先 (Mobile-First)：** 針對行動端流量進行極致優化。
- **極速載入 (Performance)：** 優化代碼結構與圖片資源。
- **現代美學 (Modern UI)：** 使用 Tailwind CSS 與 Framer Motion 打造精品質感。
